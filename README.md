## Overview

* C fundamentals → C++ → OOP → STL
* Concept-driven with hands-on practice
* Ends with mini projects

---

## Curriculum

<details>
<summary><strong>Core Topics</strong></summary>

* C: syntax, control flow, arrays, pointers, memory
* C++: references, I/O, overloading
* OOP: classes, inheritance, polymorphism
* STL: vector, map, set, algorithms

</details>

<details>
<summary><strong>Progression</strong></summary>

* Foundation → Object-oriented design → STL + Projects

</details>

## Practice

<details>
<summary><strong>What You Do</strong></summary>

* Solve logic problems
* Work with recursion and pointers
* Build class-based systems
* Use STL for efficient solutions

</details>

## Outcomes

<details>
